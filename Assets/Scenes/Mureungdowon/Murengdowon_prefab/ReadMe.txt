SETUP:
These models are set up to use a tiled texture which are located in
the Textures folder. You can edit the resoultion by changing the
X and Y tile settings. 

UNITY3D:
You can import these models into any 3D editor but they were
intended for the use of Game Design in Unity3D. You can open
our example scene with our HeavenModular.unitypackage

SHADERS:
1) The windows need a double sided window shader
2) The fountain water needs an animated/flowing shader
3) The platform ground needs a grass shader (if wanted)
4) The platform mana was intended to be placed under 
platforms as a floating energy FX
